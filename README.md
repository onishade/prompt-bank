## 🚀 Cara Pakai

1. Pilih kategori konten.
2. Ganti placeholder `[PROJECT]`, `[TOPIC]`, `[TESTNET]` sesuai kebutuhan.
3. Masukkan ke LLM/AI favoritmu.
